🧠 Sobre o Projeto

Este projeto realiza uma análise de dados focada em cancelamento de clientes (churn), buscando entender por que os clientes deixam o serviço.

A partir de uma base em Excel, foram analisados comportamentos, padrões e características dos clientes para gerar insights estratégicos.

🎯 Objetivo

Identificar padrões que influenciam o cancelamento e responder perguntas como:

❓ Quais clientes têm maior chance de cancelar?

❓ O tipo de plano influencia no churn?

❓ O suporte impacta na decisão do cliente?

❓ Existe relação entre atraso e cancelamento?

🔍 Principais Insights

✨ Descobertas importantes:

📉 Planos mensais têm maior taxa de cancelamento

📞 Muitos contatos com suporte = maior chance de churn

⏳ Atrasos elevados aumentam significativamente o cancelamento

⚠️ Taxa de churn identificada: ~56%
